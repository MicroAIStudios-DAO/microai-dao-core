# MicroAI Studios Documentation

Welcome to the official documentation for MicroAI Studios. This repository contains all the technical, legal, and marketing materials related to our AI-augmented corporate governance platform.

## Documentation Structure

Our documentation is organized into the following sections:

*   **[Products](products/):** Detailed information about our SaaS products, including ExecAI, AIIntegrationCourse, and RET.
*   **[Technical](technical/):** Technical whitepapers, architecture diagrams, and smart contract implementation guides.
*   **[Legal](legal/):** Legal documents, including our Wyoming DAO LLC articles of organization and operating agreement.
*   **[Marketing](marketing/):** Marketing materials, including our ecosystem launch strategy.
*   **[Testing](testing/):** Test reports and summaries.
